Upload Foto Profil - Tanpa JavaScript

D
JPG, PNG, GIF, WEBP (Max: 2MB)

Cara Kerja (Tanpa JavaScript):

  1. User pilih foto dengan file input
  2. User klik "Simpan Perubahan"
  3. Form di-submit ke server
  4. Server proses upload foto
  5. Server redirect kembali dengan foto baru

Keuntungan: Tidak ada masalah kompatibilitas, bekerja di semua browser, lebih stabil
